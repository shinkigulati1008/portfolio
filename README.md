# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### `npm test`

Launches the test runner in the interactive watch mode.\
See the section about [running tests](https://facebook.github.io/create-react-app/docs/running-tests) for more information.



### Packages Installed

@emailjs/browser:

This package provides functionality for sending emails directly from the browser using EmailJS.
Useful if you want to incorporate email functionality within your app.

@fortawesome/fontawesome-svg-core, @fortawesome/free-brands-svg-icons, @fortawesome/free-solid-svg-icons, and @fortawesome/react-fontawesome:
These packages are part of the Font Awesome icon library.
You can use them to easily include icons (both solid and brand icons) in your application.
For example, you can add social media icons or other graphical elements.

@testing-library/jest-dom, @testing-library/react, and @testing-library/user-event:
These packages are essential for testing your React components.
They provide utilities for writing unit tests and interacting with components during testing.

animate.css:
Animate.css is a library of cross-browser CSS animations.
You can use it to add smooth animations to your UI elements, including modals.

firebase:
Firebase is a powerful backend-as-a-service (BaaS) platform.
It offers authentication, real-time database, cloud storage, and more.
Useful for handling user authentication or storing data related to your modal.

gsap-trial:
GSAP (GreenSock Animation Platform) is a popular animation library.
It allows you to create complex animations and transitions.
You can use it to animate your modal’s entrance, exit, or any other interactions.

leaflet and react-leaflet:
Leaflet is a lightweight mapping library for interactive maps.
React-Leaflet provides React components for integrating Leaflet maps into your app.
If your modal involves maps or location-based features, these packages are handy.

loaders.css and react-loaders:
Loaders.css provides a collection of loading animations (spinners).
React-Loaders is a React wrapper for using these loaders.
Great for displaying loading spinners while fetching data or performing actions.

react-router-dom:
React Router is a popular library for handling routing in React applications.
react-router-dom specifically provides components for web routing.
Useful if your modal needs to navigate between different views.

react-scripts:
react-scripts is a set of scripts and configuration used by Create React App (CRA).
It handles development server, build process, and other tasks.
Essential for running and building your React app.

sass:
SASS (Syntactically Awesome Style Sheets) is a CSS preprocessor.
It allows you to write more maintainable and modular CSS.
Useful for styling your modal components.

web-vitals:
Web Vitals is an initiative by Google to measure and improve web performance.
It provides tools for tracking metrics like loading time, interactivity, and visual stability.
Useful for optimizing your modal’s performance.
